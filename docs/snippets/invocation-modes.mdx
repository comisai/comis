Pick whichever matches how you installed Comis:

<Tabs>
  <Tab title="Installed (recommended)">
    ```bash
    comis <command>
    ```

    This works after installing via `npm install -g comisai` or the one-line install script.
  </Tab>
  <Tab title="From source">
    ```bash
    node packages/cli/dist/cli.js <command>
    ```

    Use this if you cloned the repository and are running Comis directly from source.
  </Tab>
</Tabs>

<Tip>
The install script sets up the `comis` command automatically -- you almost certainly want the Installed tab.
</Tip>
