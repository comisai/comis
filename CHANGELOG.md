### Added
- Initial public open-source release

- Support for Discord, Telegram, Slack, WhatsApp, Signal, iMessage, IRC, LINE
- Multi-agent architecture with DAG pipelines
- Persistent semantic memory with trust partitioning
- 50+ tool integrations with MCP support
- 17 security layers including OS-level sandbox
- 7-layer context engine with prompt cache optimizations
- Three-tier budget guard for cost control
- HTTP, JSON-RPC, and WebSocket gateway
- CLI and web dashboard
